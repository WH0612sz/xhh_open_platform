快速开始

  1. 注册并创建应用，获取 AppID / Secret
  2. 调用 Token 接口，获得访问凭证
  3. 调用核心 API（创建订单 / 查询项目）
  4. 配置回调并验证结果
POST /v1/auth/token
{
  "app_id": "你的AppID",
  "app_secret": "你的Secret"
}

鉴权与签名

所有请求需携带 Token 与签名，签名规则包含 body + timestamp + nonce。

回调配置

在控制台配置回调地址，接收支付/发放等事件通知。

错误码

错误码说明建议
4001签名校验失败检查签名规则与时间戳
4010Token 失效重新获取 Token
5001内部错误稍后重试或提交工单
