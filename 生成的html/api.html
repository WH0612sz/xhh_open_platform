接口：创建捐赠订单

适用能力：捐赠能力 · 权限范围：donation.write · 环境：正式/沙箱

POST /v1/donation/order

请求信息

MethodPOST
Content-Typeapplication/json
鉴权Token + 签名

请求参数

字段类型必填说明
project_idstring是项目编号
amountint是金额（分）
order_nostring是合作方订单号
notify_urlstring是支付回调地址

请求示例

curl -X POST https://oapi.gongyi.qq.com/v1/donation/order \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "2026A001",
    "amount": 1000,
    "order_no": "ORD20260222001",
    "notify_url": "https://partner.com/callback"
  }'

响应示例

{
  "code": 0,
  "message": "success",
  "data": {
    "order_id": "GY20260222088",
    "pay_url": "https://pay.gongyi.qq.com/xxx"
  }
}

错误码

错误码说明建议
4001签名错误检查签名与时间戳
4020权限不足申请 donation.write 权限
